## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org) (v14 or above)
- [Yarn 1](https://classic.yarnpkg.com/en/docs/install)

### Installation

1. Clone the repository

```sh
git clone https://github.com/chingu-voyages/v29-bears-team-07.git
```

2. Install dependencies

```sh
yarn install
```
